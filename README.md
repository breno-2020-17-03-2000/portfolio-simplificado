Portfólio Simplificado é uma landing page feita usando as principais linguagens de programação Front-end HTML, CSS e o JavaScript, esse projeto
trás basicamente um compilado de todas as minhas experiências e projetos dentro da programação até aqui, logo a idéia foi se utilizar das 
três principais linguagens de programação Front-end para linkar todos os meus projetos em uma única página adaptada as minhas preferências.
